Settings

Buzzer Volume
How loud shoud vario beep? (1-2 square is good for helmet mounting, 3 when the wind is high, 4 when you mount the vario on harness or cockpit). You can change this via button menu.
Auto power-off
How long shoud vario stay on when there is no change in altitude? Set to zero to disable. You can't change this via button menu.
Selected profile
What profile do you want to use? You can change this using button after power-on sequence.
Lift Tresholds





Here you can change the lift treshold presets. You can't change values via button menu, however you can select diffrent preset.
Sink Tresholds





Here you can change the sink treshold presets. You can't change values via button menu, however you can select diffrent preset.

Experimental (beta2 only)

Silent start
This option will suppress sound output until the filter is stabilised.
Fluid beep
Continous lift beep frequency update.
